# GIT RepoFinder 🔍

An interactive and responsive web tool to search and display GitHub repositories by username using the GitHub REST API.

## 🚀 Features

- 🔎 Search GitHub repositories by any valid username  
- 📦 Displays key repository details: name, description, stars, forks  
- 💡 Error handling for invalid usernames or no repositories  
- 🎨 Clean and responsive UI using Bootstrap  
- 🖥 Fully functional in modern browsers  

## 🛠 Tech Stack

- *Frontend*: HTML, CSS, Bootstrap  
- *Logic*: JavaScript (ES6)  
- *API*: GitHub REST API v3
